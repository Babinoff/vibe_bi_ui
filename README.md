# BIUI - Automated Business Intelligence Tool

## Описание проекта
Приложение **BIUI** предназначено для автоматизированного создания бизнес-аналитики с использованием нодового интерфейса и возможностей искусственного интеллекта. Система позволяет пользователям загружать данные, трансформировать их с помощью ИИ-генерируемых скриптов и создавать интерактивные визуализации.

## Основные компоненты интерфейса

### 1. Панель данных (Data Source)
- **Расположение**: Левая верхняя часть экрана.
- **Функционал**:
  - Загрузка файлов (MVP: CSV; в планах: JSON, XML).
  - Предпросмотр данных: Узкая вертикальная полоса под кнопкой загрузки.
  - Управление: Скролл по ширине/высоте, сворачивание/разворачивание панели.
  - **Точка подключения**: Выходной порт для передачи "сырых" данных в нодовое поле.

### 2. Рабочее поле (Node Canvas)
Основное пространство приложения для создания логики обработки данных.
- **Типы нод**:
  - **Ноды преобразования**: Конвертация, фильтрация, агрегация данных.
  - **Ноды отображения**: Визуализация данных (графики, диаграммы).

#### Конструкция Ноды
Каждая нода имеет унифицированную структуру:
- **Порты**:
  - Входы: `IN[0]`, `IN[1]` и т.д.
  - Выходы: `OUT[0]`, `OUT[1]` и т.д.
- **Блок ИИ-генерации**:
  - **Промт-поле**: Ввод задачи на естественном языке (например, "Отфильтруй продажи выше 1000").
  - **Кодовое поле**: Отображение сгенерированного кода (Python/JS).
  - **Управление**: Кнопка "Play" для выполнения, индикация успеха/ошибки.
  - **Уточнение**: Возможность добавлять цепочки промтов ("+" рядом с основным) для корректировки кода или указания конкретных строк.
- **Превью (Скрываемое)**:
  - Нижняя часть ноды.
  - Отображает результат выполнения: трансформированную таблицу или миниатюру графика.

### 3. Визуализация
- Возможность указания библиотек визуализации (путь к GitHub, npm, python-библиотеке).
- ИИ генерирует код для конкретной библиотеки и типа графика.

### 4. Презентационный слой (Dashboard)
- **Расположение**: Скрытая панель в правой части экрана (аналог панели загрузки).
- **Итоговая нода**:
  - Принимает данные из любых `OUT` портов рабочих нод.
  - При клике разворачивает интерфейс конструктора дашборда.
- **Конструктор**:
  - Масштабируемые прямоугольные области для каждого входящего графика/данных.
  - Свободное размещение и изменение размеров (Drag & Drop, Resize).
  - Добавление текстовых подписей и заголовков (из метаданных ноды или вручную).

## Общие этапы реализации

### Этап 1: Каркас приложения и управление данными
1. **Инициализация проекта**: Настройка стека (React, TypeScript, State Management).
2. **UI Layout**: Реализация основной сетки (Левая панель, Холст, Правая панель).
3. **Загрузчик данных**:
   - Реализация компонента загрузки CSV.
   - Парсинг данных на клиенте.
   - Компонент предпросмотра (Data Grid) с возможностью сворачивания.

### Этап 2: Нодовый движок (Core)
1. **Интеграция библиотеки нод**: (например, React Flow).
2. **Базовая структура ноды**: Верстка кастомной ноды с портами, полями ввода и кнопками.
3. **Менеджмент связей**: Логика передачи данных от `OUT` к `IN`.
4. **Изолированная среда выполнения**: Безопасное выполнение JS/Python кода (Web Workers / Pyodide).

### Этап 3: Интеграция ИИ и Генерация кода
1. **API Клиент**: Настройка взаимодействия с LLM.
2. **Генератор кода**:
   - Формирование контекста для ИИ (схема данных + промт пользователя).
   - Парсинг ответа и вставка кода в редактор ноды.
3. **Система корректировок**: Логика добавления дополнительных промтов и перегенерации частей кода.

### Этап 4: Визуализация и Библиотеки
1. **Поддержка библиотек**: Механизм динамического импорта библиотек визуализации.
2. **Рендеринг графиков**: Компоненты-обертки для отображения результатов в превью ноды.
3. **Генерация визуализаций**: Промпт-инжиниринг для создания корректных конфигов графиков.

### Этап 5: Презентационный режим
1. **Итоговая нода**: Логика сбора данных со всех подключенных входов.
2. **Конструктор дашборда**:
   - Grid-система для виджетов (React-Grid-Layout или аналог).
   - Компоненты отображения графиков в полном масштабе.
   - Инструменты добавления текста и аннотаций.
